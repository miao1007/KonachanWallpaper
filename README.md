Konachan壁纸
===================================
  这是一个可以搜索，下载高分辨率壁纸的Android程序,目前正在开发中，希望小伙伴们能够帮我共同完善这个程序.使用Andorid Studio 开发

程序原理
-----------------------------------
* 使用Konachan的API，下载JSON数据
* 解析数据，显示图片到手机上
* 注意，我这个程序去掉了R18的图片，想要实现的同学可以研究一下代码

KonachanWallpaper
===================================

-----------------------------------
